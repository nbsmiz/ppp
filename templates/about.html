About

    Webpage

      Flask - Webpage is a Flask app.

      Jinja - Jinja templating was used some to ease the wrangling of HTML.

    Analysis

      Python - All data extraction, transforming, and analyzing was done with Python or a library thereof.

      Pandas - The majority of data cleaning, restructuring and analysis was done in Pandas.

      Pickle - Pickle was used to save reduced Pandas dataframes and other objects for later reuse.

    Mapping

      Folium - The Folium library was used to produce the maps, the loan markers and the HTML used to render the map.

      Geocoder - The Geocoder library was used to pull latitude and longitude for loan recipients from OpenStreetMap API.

      Leaflet.js - Folium produces Leaflet.js maps, but some additional work with this was needed to add the search functionality.

    Visualization

      Plotly - The Plotly Python Library was used to produce aggregate data charts from Pandas dataframes.
